Overview of the EDR Architecture

07.07.5050 10:00
TLDR ; Endpoint Detection and Response is a defensive product. It is simply an application that runs on target workstations or servers on which it collects data on the security environment called telemetry. In this article, I try to explain the methods EDRs use to collect these information. This is based on an amazing book called Evadin EDR by Matt Hand. 1- Components of an EDR The Agent The EDR agent is an application that manages and analyzes data from various sensors to identify activities that might indicate a security threat.
